# gameoflife-1d

One-dimensional Game of Life for the terminal

## Requirements

You only need the g++ compiler and these C++ libraries:

* iostream
* cstdlib
* cstring
* vector
* boost/dynamic_bitset.hpp

## Installation

From the root folder, just call:

    make

Make sure the shell script is executable:

	chmod +x gol1d.sh

## Usage

Run it like this:

	./gol1d.sh
